# Credit Card Approval Prediction

## Project Overview

This project predicts whether a customer's credit card application will be approved using Machine Learning.

The application compares multiple machine learning algorithms and automatically selects the best-performing model.

---

## Algorithms Used

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Installation

```bash
pip install -r requirements.txt
```

Run preprocessing

```bash
python preprocessing.py
```

Train models

```bash
python train_logistic.py
python train_decision_tree.py
python train_random_forest.py
python train_xgboost.py
```

Evaluate

```bash
python evaluate_model.py
```

Run Flask

```bash
python app.py
```
